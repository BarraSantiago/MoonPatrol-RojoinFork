Mad Patrol

Reimagination of original 1982 Moon Patrol

Controls:

Player 1:

AD = Move left and right
Space = Jump
W = Shoot
F = Shoot only for singleplayer

Player 2:

Left and right arrow keys = Move left and right
Up arrow key = Jump
right Ctrl = Shoot
